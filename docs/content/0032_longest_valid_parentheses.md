```python
def longest_valid_parentheses(s):
    stack = [-1]
    max_len = 0
    for i, ch in enumerate(s):
        if ch == "(":
            stack.append(i)
        else:
            stack.pop()
            if not stack:
                stack.append(i)
            else:
                max_len = max(max_len, i - stack[-1])
    return max_len


s = "(()"  # 2 
# s = ")()())"  # 4
result = longest_valid_parentheses(s)
print(result)
```

这道题目（LeetCode 32. 最长有效括号）是括号匹配系列中的**终极挑战**。如果说“有效括号”只是检查是否合法，那么这一题就是在断壁残垣中寻找**最长的一段“太平盛世”**。

它的核心思想是 **“基准锚点（Reference Anchoring）”**，利用栈来记录“上一次不合法的位置”。

---

### 一、 核心思想：寻找“分界线”

在一段括号字符串中，一段有效的括号总是被“不合法的括号”隔开的。
1.  **左括号 `(`**：是希望的起点，我们把它藏在栈里，等待救赎。
2.  **右括号 `)`**：是现实的重击。如果它能配对，我们就计算长度；如果它不能配对，它就成了新的**“禁区边界”**。
3.  **哨兵机制**：为了方便计算长度，我们先在栈底放一个 `-1`。它就像是一个虚拟的墙，告诉我们：如果第一个字符就开始匹配，长度应该从哪里算起。



---

### 二、 算法逻辑：栈中的“边界守卫”

#### 1. 初始化哨兵
* `stack = [-1]`：这个 `-1` 极其关键。它代表**最后一个没有被匹配的右括号的下标**。

#### 2. 遍历规则
* **遇到左括号 `(`**：
    * 不多废话，直接把下标 `i` 压入栈。这代表一个潜在的合法序列起点。
* **遇到右括号 `)`**：
    * **先弹出栈顶**：尝试匹配掉最近的一个左括号（或者是哨兵）。
    * **判断栈是否为空**：
        * **栈空了**：说明当前的右括号是个“显眼包”，它没有左括号可以匹配了。它成了新的**分割点**，我们把它入栈，作为后续计算长度的新基准。
        * **栈不空**：说明匹配成功！当前的有效长度就是 `当前下标 i - 现在的栈顶下标`。

#### 3. 长度更新
* `max_len = max(max_len, i - stack[-1])`：每次匹配成功，我们都刷新一下最长记录。

---

### 三、 过程模拟 (`s = ")()())"`)

1.  **初始**：`stack = [-1]`
2.  **i=0, `)`**：弹出 -1，栈空。入栈 0。`stack = [0]`（0成了新基准）。
3.  **i=1, `(`**：入栈 1。`stack = [0, 1]`。
4.  **i=2, `)`**：弹出 1，栈不空。`max_len = max(0, 2 - 0) = 2`。
5.  **i=3, `(`**：入栈 3。`stack = [0, 3]`。
6.  **i=4, `)`**：弹出 3，栈不空。`max_len = max(2, 4 - 0) = 4`。
7.  **i=5, `)`**：弹出 0，栈空。入栈 5。`stack = [5]`。
8.  **结果**：4。

---

### 四、 复杂度分析

* **时间复杂度：$O(N)$**
    * 无论字符串多复杂，每个下标只进出栈一次，线性扫描。
* **空间复杂度：$O(N)$**
    * 最坏情况下（全是左括号），栈的大小会达到 $N$。

---

### 五、 总结金句

> **“这叫‘步步为营，界限分明’。在这里，栈不再仅仅是存放期许的容器，它更像是一个记录边界的刻度尺。通过预设一个‘哨兵’，我们将复杂的嵌套和并列关系，简化为了简单的下标相减。每一个留存在栈底的元素，都是一段合法文明的‘边疆’。这种寻找基准的思想，是解决区间长度问题的高端技巧。”**

### 💡 进阶思考
这道题有一个惊艳的 **$O(1)$ 空间解法**：从左往右扫一遍计数，再从右往左扫一遍计数。你知道为什么必须扫**两遍**才能确保万无一失吗？（提示：想想 `((()` 这种情况，只从左往右扫会漏掉什么？）
