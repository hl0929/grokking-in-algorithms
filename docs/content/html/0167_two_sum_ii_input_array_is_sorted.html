在有序数组两端建立指针。左指针指向最小值，右指针指向最大值。

Two Sum Logic O(N) Time | O(1) Space
left, right = 0, len(numbers) - 1
while left < right:
total = numbers[left] + numbers[right]
if total == target:
return [left + 1, right + 1]
elif total > target:
right -= 1
else:
left += 1
