QUICK SORT 算法分析

点击播放查看分治过程。我们将数组第一个元素作为基准值，将其他元素分为比它小、等于它、比它大三部分。

Recursive Logic Average O(N log N)
def _quick_sort(nums):
if len(nums) < 2:
return nums
mid = nums[0] # Pivot
left = [i for i in nums if i < mid]
middle = [i for i in nums if i == mid]
right = [i for i in nums if i > mid]
return _quick_sort(left) + middle + \
_quick_sort(right)

分治策略

1. 基准： 选择第一个元素作为 Pivot。
2. 分区： Left (<), Middle (==), Right (>)。
3. 递归： 对 Left 和 Right 继续快排。
