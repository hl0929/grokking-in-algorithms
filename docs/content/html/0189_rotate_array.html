轮转数组可以通过三次反转在原地、O(1) 空间内完成。首先反转整个数组，然后反转前 k 个，最后反转剩余部分。

Rotation Logic O(N) Time | O(1) Space
def rotate(nums, k):
n = len(nums)
k %= n
# 1. 反转全数组
reverse(0, n - 1)
# 2. 反转前 k 个
reverse(0, k - 1)
# 3. 反转后 n-k 个
reverse(k, n - 1)
def reverse(start, end):
while start < end:
nums[start], nums[end] = nums[end], nums[start]
start += 1; end -= 1
