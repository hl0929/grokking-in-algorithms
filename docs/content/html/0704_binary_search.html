准备搜索。由于数组已排序，我们可以通过每次排除一半的元素来快速锁定目标。

Binary Search Logic O(log N) Time
def search(nums, target):
left, right = 0, len(nums) - 1
while left <= right:
mid = (left + right) // 2
if nums[mid] == target:
return mid
if nums[mid] > target:
right = mid - 1
else:
left = mid + 1
return -1

算法要点

1. 前提：数组必须是有序的。
2. 核心：通过 mid 将搜索范围减半。
3. 停止：找到目标或 left > right。
