prev2 = nums[0] # 第1家 prev1 = max(nums[0], nums[1]) # 前2家 for i in range(2, len(nums)): curr = max(prev1, prev2 + nums[i]) prev2 = prev1 prev1 = curr return prev1

对于每一家房子，我们都在权衡：“如果偷这一家，就要放弃前一家” 还是 “直接沿用偷完前一家的最优解”。
