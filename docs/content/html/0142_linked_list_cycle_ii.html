def detectCycle(head): slow = fast = head while fast and fast.next: slow = slow.next fast = fast.next.next if slow == fast: break # 相遇 else: return None ptr = head while ptr != slow: ptr = ptr.next slow = slow.next return ptr # 环入口

相遇后，从头节点和相遇点同步出发，再次相遇处即为环入口。
