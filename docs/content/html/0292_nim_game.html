ANALYSIS 博弈状态分析

点击“开始演示”查看博弈过程。关键在于：只要石头数不是 4 的倍数，先手总能通过拿走一部分，让剩余石头变成 4 的倍数留给对方。

Logic Code O(1) Solution
def canWinNim(n):
# 如果 n 是 4 的倍数
# 无论你拿 1, 2, 3 个
# 对手总能拿走剩下的部分凑成 4
return n % 4 != 0

关键点提示

  • 你是先手，每次可拿 1-3 块。

  • 拿到最后一块石头的人获胜。

  • 每 4 个石块组成一个“必胜循环”。
