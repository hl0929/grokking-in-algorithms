零钱兑换算法可视化

核心思想：dp[i] = min(dp[i], dp[i - coin] + 1)

可用硬币:
dp 数组（索引 i 代表金额，值代表最少硬币数）
正在初始化 dp[0] = 0...
目标金额: 11, 初始化数组为 Infinity
Coin Change Logic STEP 0
dp = [float("inf")] * (amount + 1) dp[0] = 0 for i in range(1, amount + 1): for coin in coins: if i >= coin: dp[i] = min(dp[i], 1 + dp[i-coin]) return dp[amount] if ... else -1
