```python
def max_sliding_window(nums, k):
    if not nums or k == 0:
        return []
    from collections import deque
    dq = deque()
    res = []
    for idx, num in enumerate(nums):
        while dq and nums[dq[-1]] < num:
            dq.pop()
        dq.append(idx)
        if dq[0] <= idx - k:
            dq.popleft()
        if idx >= k - 1:
            res.append(nums[dq[0]])
    return res


nums = [10, 2, 3, 1] 
k = 2 # [10,3,3]
result = max_sliding_window(nums, k)
print(result)
```

这道题目（LeetCode 239. 滑动窗口最大值）的核心思想是 **“单调队列（Monotonic Queue）”**。这道题更像是在维护一个 **“实时更新的强力竞争榜”**。

通过单调队列，我们可以把找区间最大值的复杂度从 $O(N \times K)$ 降低到惊人的 **$O(N)$**。

---

### 一、 核心思想：长江后浪推前浪

在一个滑动的窗口里，如果新进来的数比前面的数大，那么前面的数就**永远不可能**再成为最大值了（因为新数更强，且存活时间更长）。

1.  **单调递减性**：我们维护一个队列，里面只存可能成为最大值的候选人的**索引**。这个队列里的数值必须是**从大到小**排列的。
2.  **及时退役**：
    * **被踢出（后浪推前浪）**：当新元素进来时，把队列后端所有比它小的元素全部踢掉。
    * **过期（老兵凋零）**：当队列头部的元素已经滑出窗口范围时，将其移出。
3.  **榜首即答案**：队列的第一个元素（头部）永远是当前窗口最大的那个。

---

### 二、 算法逻辑：三步走

以 `nums = [10, 2, 3, 1], k = 2` 为例：

#### 1. 淘汰弱者：保持单调
* 每处理一个 `num`，观察队列尾部。
* 如果尾部对应的数比 `num` 小，直接 `pop` 掉。
* *含义：既然有了更强更年轻的 `num`，那些又弱又老的数就没机会出头了。*

#### 2. 检查过期：维持窗口
* 检查队列头部的索引。如果它已经小于等于 `idx - k`，说明它已经不在当前的 $k$ 长度窗口里了。
* **动作**：从左端弹出 `popleft`。

#### 3. 记录榜首
* 只要窗口已经成型（`idx >= k - 1`），队列头部的那个数就是我们要的最大值。



---

### 三 : 过程模拟 (`nums = [10, 2, 3, 1], k = 2`)

1.  **处理 10** (idx 0)：
    * 队列为空，直接进：`dq = [0]`。
    * 窗口未成型。
2.  **处理 2** (idx 1)：
    * 2 比 10 小，不踢。进：`dq = [0, 1]`。
    * 窗口成型，榜首是 `nums[0] = 10`。结果 `res = [10]`。
3.  **处理 3** (idx 2)：
    * **踢人**：3 比队尾的 2 大，2 被踢走。
    * **过期检查**：头部的 10 (idx 0) 刚好滑出窗口（$0 \le 2-2$），10 被弹出。
    * **入队**：3 进来。`dq = [2]`。
    * 榜首是 `nums[2] = 3`。结果 `res = [10, 3]`。
4.  **处理 1** (idx 3)：
    * 1 比 3 小，不踢。进：`dq = [2, 3]`。
    * 过期检查：头部的 3 (idx 2) 还在窗口内。
    * 榜首是 `nums[2] = 3`。结果 `res = [10, 3, 3]`。

---

### 四、 复杂度分析

* **时间复杂度：$O(N)$**
    * 尽管有嵌套的 `while` 循环，但每个元素的索引最多只进队一次、出队一次。总操作数是 $2N$ 级别的。
* **空间复杂度：$O(K)$**
    * 单调队列中最多同时存放 $K$ 个元素的索引。

---

### 五、 总结金句

> **“单调队列就像是一个极度现实的‘选秀榜单’：每一个新人进来，都会把挡在前面且不如自己强的老人全部‘卷’走。而那些即便很强但已经‘超龄’（滑出窗口）的前辈，也会被强制退役。这保证了留在榜首的永远是当前武林中最强的霸主。”**
